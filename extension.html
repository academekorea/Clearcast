Install in under 2 minutes

No Chrome Web Store required. Load directly as a developer extension.

1
Download the extension files
Click the download button above. A ZIP file (podlens-extension-v1.1.0.zip) will save to your Downloads folder.
2
Unzip the file
Double-click the ZIP to extract it. You’ll get a folder called podlens-extension.
3
Open Chrome Extensions
In Chrome, go to chrome://extensions — or Menu → Extensions → Manage Extensions.
4
Enable Developer Mode
Toggle the Developer mode switch in the top-right corner of the Extensions page.
5
Load the extension
Click Load unpacked and select the podlens-extension folder you unzipped. The Podlens icon will appear in your toolbar immediately.
Submitting to the Chrome Web Store
We’re submitting to the Chrome Web Store. Until approval, install manually using the steps above — it takes under 2 minutes and works identically.
